# File System Components

This module contains components for PowerShell Universal Apps for working with the file system.

## Components

### `New-UDFileSystemBrowser`

This component provides a file system browser that allows users to select files and folders. 

```powershell
New-UDFileSystemBrowser -PublishedFolder 'Repository'
```
